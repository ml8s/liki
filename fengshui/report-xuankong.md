# 玄空风水报告模板

基于对应 API 返回数据生成玄空报告。

## 数据来源

字段含义参考 `rpc.discover` 返回的 schema。只引用实际存在的字段，不编造。

## 报告结构

### 一、三元九运

当前所处元运（上/中/下元，九运中哪一运），起止年份，当运当旺之星。

### 二、飞星盘

九宫飞星盘（运盘+山盘+向盘），每宫三星组合解读。重点：旺山旺向/上山下水/双星会向/双星会坐判断。

### 三、九宫逐宫分析

每宫三星组合（山/向/运）的五行生克关系，吉凶含义。

### 四、财位与文昌

当运财星所在方位，文昌位，忌星方位。

### 五、布局与化解

催旺吉星、化解凶星的具体方法（五行化煞、流年飞星叠加注意事项）。
