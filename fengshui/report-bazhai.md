# 八宅风水报告模板

基于 `bazhai.minggua` 和 `bazhai.chart` 返回数据生成八宅报告。只引用实际存在的字段，不编造。

## 数据来源

字段含义参考 `rpc.discover` 返回的 schema。只引用实际存在的字段，不编造。

## 报告结构

### 一、命卦判定

出生年份 → 命卦（乾/坤/震/巽/坎/离/艮/兑）→ 东四命或西四命。

### 二、八宅方位

四吉方（生气/延年/天医/伏位）、四凶方（祸害/六煞/五鬼/绝命）在各方位分布。

### 三、布局建议

- 大门、主卧、厨房宜在吉方
- 卫生间、储物间宜在凶方
- 命卦与宅卦是否配合

### 四、吉凶化解

凶方化解方法（五行制化、屏风/植物/颜色调整）。
