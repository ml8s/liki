# Liki Skills

灵机（Liki）AI 命理顾问 — 技能包合集。

## 安装

```bash
# 全家桶（全部功能）
npx skills add <repo-url>

# 单个技能
npx skills add <repo-url>/naming      # 起名
npx skills add <repo-url>/bazi        # 八字
npx skills add <repo-url>/ziwei       # 紫微
npx skills add <repo-url>/ask         # 问事（奇门/六爻/黄历）
npx skills add <repo-url>/fengshui    # 风水（八宅/玄空）
```

## 技能列表

| 技能 | 说明 |
|------|------|
| [naming](./naming/) | AI 起名顾问，结合八字用神与五格三才 |
| [bazi](./bazi/) | 八字命理，排盘/合盘/大运/流年 |
| [ziwei](./ziwei/) | 紫微斗数，十二宫星曜四化 |
| [ask](./ask/) | 占卜择日，奇门遁甲/六爻/黄历 |
| [fengshui](./fengshui/) | 风水分析，八宅命卦/玄空飞星 |
