## P1 — 低投入高收益

### 1. 斯多葛语气（去吉凶标签）

**目标**：将输出中的评价性措辞（吉/凶/好/坏/顺利/坎坷）替换为能量状态描述。

#### 修改范围

| 文件 | 修改内容 |
|------|---------|
| `mingli/SKILL.md` 输出规则 | 加一条语气规则 |
| `mingli/report-chart.md` 综合建议 | 四维度措辞去评价标签 |
| `wengua/SKILL.md` 输出规则 | 现有"结论先行"后补语气规则 |
| `wengua/report-liuyao.md` | 断语中的吉凶描述改为方向描述 |
| `wengua/report-huangli.md` | **不改** – 宜/忌是黄历术语，保留 |
| `mingli/SKILL.md` 分析要点 | "结合用神判断吉凶"改为"判断能量走向" |
| `qiming/SKILL.md` 选字原则 | 避免"好名字"等评价措辞 |
| `/fengshui/report-bazhai.md` | **不改** – 吉方/凶方是风水术语，保留 |

#### 措辞映射表

| 禁用 | 替换为 | 适用场景 |
|------|--------|---------|
| 吉利、吉 | 能量通畅、顺势、支撑有力 | 八字用神到位、流年生扶 |
| 凶、不吉 | 能量受阻、需调整、方向不对 | 用神受克、流年冲刑 |
| 好、不错 | 状态积极、释放期 | 综合建议领域描述 |
| 不好、差 | 收敛期、调整期 | 综合建议领域描述 |
| 顺利 | 能量走向顺畅 | 大运流年分析 |
| 坎坷 | 张力较大、需反复打磨 | 大运流年分析 |
| 宜做某事 | **保留术语，不替换** | 黄历宜/忌是领域固有格式 |
| 不宜做某事 | **保留术语，不替换** | 黄历宜/忌是领域固有格式 |

#### 分支处理

| 场景 | 处理 | 示例 |
|------|------|------|
| 用户直接问"这事吉凶如何" | 仍给方向判断，但用"能量状态"措辞 | "此事当前能量状态为**受阻**，关键在XX月后才有转机" |
| 用户问运势 | 趋势描述，不用评价 | "今年事业方向处于**释放期**，官星能量外显" |
| 风水方位（八宅） | **保留原术语** | 四吉方/四凶方是领域术语 |
| wengua 结论先行 | 改为"方向判断"而非"吉凶判断" | "卦象显示此事**可行**"（不变）→ 本身已经 OK |
| 从格 | 方向按从格走势描述 | "顺势而行，能量集中在XX领域" |

### 2. 同名陷阱检测

**目标**：综合命理（八字+紫微同时分析）时，防止 LLM 混淆不同体系的术语。

#### 修改范围

`mingli/SKILL.md` 分析要点加一条：

```
## 分析要点

- 八字：以用神为核心...
- 合会冲刑：抓关键关系...
- 紫微：命宫为体...
- 大运流年：...
- **同名陷阱**：八字与紫微术语同名不同源时须区分（如下表），不可混用推断
```

#### 陷阱速查表（写入规则）

| 术语 | 八字含义 | 紫微含义 | 是否同源 |
|------|---------|---------|:-------:|
| 七杀 | 十神（克我同阴阳） | 星曜名 | ❌ 不同源 |
| 太阳 | 五行属性（丙火） | 星曜名 | ❌ 不同源 |
| 太阴 | 五行属性（癸水） | 星曜名 | ❌ 不同源 |
| 午火/午 | 地支（藏丁火） | 午宫位置 | ⚠️ 参照系不同 |
| 火星 | — | 煞星 | ❌（八字无此术语） |
| 文昌 | 神煞（年干查） | 辅星 | ⚠️ 同名但规则不同 |
| 天魁/天钺 | 神煞 | — | ❌ 只在八字出现 |
| 印旺 | 印星十神强 | 天梁庙旺 | ✅ 同源（保护力） |
| 食伤旺 | 食神伤官强 | 文昌/文曲庙旺 | ✅ 同源（才华发散） |
| 财星强 | 正财偏财透 | 武曲/化禄庙旺 | ✅ 同源（财富能量） |

#### 分支处理

| 场景 | 处理 |
|------|------|
| 只排八字 | 不触发，跳过 |
| 只排紫微 | 不触发，跳过 |
| 综合命理（八+紫） | 必须检查 |
| 合盘 | 如双方都排了八字+紫微才触发 |
| 有歧义不确定 | 宁可分别陈述，不强拉关联 |

## P2 — 中等投入

### 3. 人格画像（解释器校准）

**目标**：从命盘结构推断用户认知模式，写入综合建议。

#### 修改范围

`mingli/report-chart.md` 综合建议第 0 段（事业之前）：

```
### 七、综合建议

**人格画像**：基于命盘信号推断——[一句话人格特征描述]

- **事业**：...
- **财运**：...
- **感情**：...
- **健康提示**：...
```

#### 八字信号 → 人格特征映射

| 盘面信号 | 人格特征 | 综合建议表述 |
|----------|---------|-------------|
| 印重（双印以上透出） | 内化型：先分析再输出 | 思考链路长但深度高，适合需要沉淀周期的领域 |
| 食伤旺 | 外放型：输出即思考 | 反应快、产出型，适合高压快节奏的创意/表达环境 |
| 身弱有印护身 | 高敏感型：需独处恢复 | 感知力强但易消耗，需要规律性独处恢复周期 |
| 身弱官杀旺 | 压力驱动型：在规则下发力 | 外部框架越清晰表现越好，适合体制化环境 |
| 原局冲多 | 张力型：选择即定义 | 常驻决策张力，不是困境是机制，每次选择都在塑造自我 |
| 原局合多 | 关系型：连接释放能量 | 能量通过协作放大，适合横向沟通型角色 |
| 从格 | 顺势型：不可逆势抗压 | 跟随外部节奏发力效率最高，逆向坚持反而消耗 |

#### 分支处理

| 场景 | 处理 |
|------|------|
| 只排紫微（不排八字） | 从紫微命宫主星推断（见下） |
| 综合命理（八+紫） | 八字信号优先，紫微验证 |
| 信号冲突（如八字印重但紫微太阳庙） | 标注复杂性，不简化 |
| 从格 | 按从格逻辑走，不套普通推断 |
| 日主中和无明显倾向 | 转向紫微命宫主星或跳过 |
| 缺少足够信息 | 跳过人格画像段 |

#### 紫微主星 → 人格特征（八字不可用时备选）

| 主星 | 特征 |
|------|------|
| 紫微 | 领导型，主动控场 |
| 天机 | 思维型，快速切换，好奇 |
| 太阳 | 外放型，慷慨直率 |
| 武曲 | 实干型，果断务实 |
| 天同 | 温和型，避冲突 |
| 廉贞 | 复杂型，内外拉扯 |
| 天府 | 保守型，稳重守成 |
| 太阴 | 内敛型，细腻敏感 |
| 贪狼 | 多才能型，适应力强 |
| 巨门 | 分析型，质疑导向 |
| 天相 | 调和型，维护秩序 |
| 天梁 | 庇护型，责任感强 |
| 七杀 | 先锋型，敢于突破 |
| 破军 | 变革型，不破不立 |
