# 起名报告模板

基于 chart 和 check 返回数据生成起名报告。只引用实际存在的字段，不编造。

## 数据来源

- `bazi.chart` → 日主、身强身弱、格局、用神/喜神/忌神
- `qiming.check` → 逐名评估：`wuge`（五格吉凶）、`sancai`（三才配置）、`phonetic`（音韵）、`wuxing`/`wuxing_match`（五行匹配）、`characters`（逐字笔画/五行/部首）

## 报告结构

### 一、命理基础与用神

日主强弱、格局、用神喜忌 → 起名五行方向（用+用或用+喜）。字数偏好及理由。

### 二、候选名字速览

每个名字一行：五行、风格定位（儒雅/刚健/灵动/古朴等）、亮点一句话。

### 三、逐个分析

每个名字按：字义（逐字解读+整体寓意）→ 典故（有则引原文出处，无则跳过）→ 五行匹配 → 三才五格 → 音韵谐音 → 实用性（生僻字/多音字/性别适配）。

### 四、横向对比与推荐

五行匹配 · 典故内涵 · 三才配置 · 五格数理 · 音韵美感 · 实用性。

综合最优推荐 + 按偏好分推：命理契合、文化内涵、音韵美感、日常实用。
