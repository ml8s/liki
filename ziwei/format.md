# 紫微斗数报告 — 基于 ziwei.chart + ziwei.judgment + ziwei.daxian 返回数据
## 数据来源
字段含义参考 `rpc.discover` 返回的 schema。只引用实际存在的字段，不编造。
## 报告结构

每节须覆盖多个分析维度，不少于以下要求：

### 一、命盘总览（不少于 3 个维度）

- 命宫主星及身宫位置
- 星曜格局（如紫府同宫、杀破狼、机月同梁等）
- 整体评级及一句话定位

### 二、身宫定位（不少于 2 个维度）

- 身宫所在宫位决定的人生着力方向
- 命宫与身宫是否协调

### 三、十二宫逐宫解读

重点解读命宫、财帛、官禄、夫妻、福德，其余宫位只标注关键异常。每宫须含主星 + 辅煞 + 一句简析。

### 四、四化分布（不少于 2 个维度）

- 化禄/化权/化科/化忌落宫及影响
- 化忌所在宫位为人生课题，须说明

### 五、三方四正

命宫、财帛、官禄、迁移四正互动。空宫借对宫星为断。

### 六、特殊格局（不少于 2 个维度）

- 格局识别和评分，多个格局时注明主格辅格
- 每个格局对事业/财运/性格的具体影响方向

### 七、大限（不少于 3 个维度）

紫微大限独立于八字大运，按命宫起运、十年一宫流转计算。每限须覆盖：

- 该限宫位的主星+四化
- 该限的人生课题（不是"顺利/坎坷"，而是具体主题）
- 与八字大运的时间对照（同步/差异）

| 年龄 | 宫位 | 主星+四化 | 主题 |
|------|------|----------|------|
| [年龄] | [宫位] | [主星+四化] | [核心课题] |
| ... | | | |
| **当前** | [宫位] | [主星+四化] | **[当前主题]** |

### 八、流年（十年详细解读）

按流年四化落原盘宫位展开，每流年须覆盖：

- 流年命宫所在宫位
- 该年对原盘的影响（哪宫被引动）
- 该年主题
