# 子平真诠 — 格局论命

## 核心原则

格局以月令为提纲，月令本气透干则定格，不透则以月令本气为虚格。格局以清纯为贵，杂浊为贱。

## 定格规则

月令支藏干透干定格局：

1. **透干优先**：月令本气→中气→余气，第一透干者定格
2. **都不透**：取月令本气为虚格，标注"格不显"
3. **多透**：以本气为主，中气余气为辅

## 成败规则

### 顺用格（官印财食）— 宜生扶保护

| 格局 | 成格条件 | 破格条件 |
|------|---------|---------|
| 正官格 | 有财生官、有印护官 | 伤官克官、官杀混杂 |
| 正印格 | 有官杀生印、比劫护印 | 财星坏印、官印中隔忌神 |
| 正财格 | 有食伤生财、有官护财 | 比劫争财 |
| 食神格 | 有比劫护食、身旺 | 枭神夺食 |

### 逆用格（杀伤劫刃）— 宜制化克泄

| 格局 | 成格条件 | 破格条件 |
|------|---------|---------|
| 七杀格 | 食神制杀、印化杀 | 杀旺无制、制杀太过 |
| 伤官格 | 印绶制伤、财星化伤 | 伤官见官 |
| 阳刃格 | 七杀制刃、食伤泄刃 | 杀刃相战无通关 |
| 建禄格 | 财官成格 | 比劫争财、食伤泄身太过 |

## 用神与相神

| 概念 | 定义 |
|------|------|
| 用神 | 命局中最关键的十神，用于维护格局平衡 |
| 相神 | 辅助用神的十神，如正官格中的印星（官印相生） |
| 喜神 | 生扶用神的五行 |
| 忌神 | 克破用神的五行 |

## 判断要点

1. **清浊**：用神一位为清，多用神或用神受冲克为浊
2. **有力无力**：用神在地支有根、得月令生扶为有力
3. **位置**：用神与日主距离越近作用越大
