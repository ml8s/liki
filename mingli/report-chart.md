# 八字排盘报告模板

基于对应 API 返回数据生成八字报告。

## 数据来源

字段含义参考 `rpc.discover` 返回的 schema。只引用实际存在的字段，不编造。

## 报告结构

### 一、命盘总览

四柱排列表（含藏干），日主五行、身强身弱、格局。

### 二、五行与十神分析

五行分布（计数+强弱）、十神配置。日主与月令关系，生克制化格局。

### 三、用神喜忌

列出扶抑、调候、格局三派各自的用神/喜神/忌神，LLM 综合三派结果给出最终用神判断及理由。

### 四、格局判定

月令透干定格——本气→中气→余气第一透干者定格，都不透则取月令本气为虚格。顺用（官印财食）生扶保护论成格，逆用（杀伤劫刃）制化克泄论成格。格局清浊高低。从格不受格局法约束，标注"从格不论格"。

### 五、大运走势

起运年龄、当前所处大运、每十年大运干支及简要提示。

### 六、当前流年

当年干支与命局关系，重点关注事项。

### 七、综合建议

**人格画像**：基于命盘信号推断——[一句话人格特征描述]

| 领域 | 命局信号 | 建议方向 |
|------|---------|---------|
| 事业 | [五行喜忌方向、当前大运的事业主题] | [可操作建议] |
| 财运 | [财星强弱、大运财星状态] | [建议方向] |
| 感情 | [夫妻宫、桃花、当前大运感情主题] | [建议方向] |
| 健康 | [五行过旺过弱对应的身体系统倾向] | [提示方向，不做医学诊断] |

用户只问单一领域时仅输出对应行。从格按从格逻辑写。命局不足以判断某领域时建议方向标注“信号不足”。
