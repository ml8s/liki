# 八字排盘报告模板

基于对应 API 返回数据生成八字报告。字段含义参考 `rpc.discover` 返回的 schema。只引用实际存在的字段，不编造。

## 数据来源

字段含义参考 `rpc.discover` 返回的 schema。只引用实际存在的字段，不编造。

## 报告结构

### 一、命盘总览

四柱排列表（含藏干），日主五行、身强身弱、格局。

### 二、五行与十神分析

五行分布（计数+强弱）、十神配置。日主与月令关系，生克制化格局。

### 三、用神喜忌

列出扶抑、调候、格局三派各自的用神/喜神/忌神，LLM 综合三派结果给出最终用神判断及理由。

### 四、大运走势

起运年龄、当前所处大运、每十年大运干支及简要提示。

### 五、当前流年

当年干支与命局关系，重点关注事项。
