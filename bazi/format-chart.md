# 八字命盘报告 — 基于 bazi.chart + bazi.judgment + bazi.yongshen 返回数据
## 数据来源
字段含义参考 `rpc.discover` 返回的 schema。只引用实际存在的字段，不编造。
## 报告结构

每节须覆盖多个分析维度，不少于以下要求：

### 一、命盘总览（不少于 3 个维度）

- 四柱排列表（含藏干）
- 日主五行、身强身弱
- 格局名称 + 一句话总评

### 二、五行与十神分析（不少于 3 个维度）

- 五行分布计数与强弱对比
- 十神配置，日主与月令关系
- 十神关键组合（杀印相生/伤官配印/财官相生等），如有则重点展开

### 三、用神喜忌（不少于 3 个维度）

- 扶抑/调候/格局三派各自的用神/喜神/忌神
- 综合三派结果，说明取舍理由（分歧时说明权重，一致时一笔带过）
- 最终用神、喜神、忌神

### 四、格局判定（不少于 4 个维度）

- 月令透干定格，虚格或正格
- 顺用或逆用，成格或破格
- 相神：辅助格局的十神是什么（如正官格的印星、七杀格的食神）
- 格局层次：清浊高低。从格标注"从格不论格"

### 五、大运走势（不少于 3 个维度）

- 起运年龄，各运干支
- 每运与命局的生克制化关系（如大运地支冲合原局哪个柱）
- 当前大运 **粗体** 标注，逐运简要提示。换运前后的衔接年份重点说明

### 六、当前流年（不少于 3 个维度）

- 当年干支与命局关系（生合冲克刑）
- 当年大运与流年的互动（大运为十年基调，流年为当年应事）
- 重点关注事项

### 七、综合建议

**人格画像**：基于命盘信号推断——[一句话人格特征描述]

| 领域 | 命局信号 | 建议方向 |
|------|---------|---------|
| 事业 | [五行喜忌方向、当前大运的事业主题] | [可操作建议] |
| 财运 | [财星强弱、大运财星状态] | [建议方向] |
| 感情 | [夫妻宫、桃花、当前大运感情主题] | [建议方向] |
| 健康 | [五行过旺过弱对应的身体系统倾向] | [提示方向，不做医学诊断] |

用户只问单一领域时仅输出对应行。从格按从格逻辑写。命局不足以判断某领域时建议方向标注"信号不足"。
