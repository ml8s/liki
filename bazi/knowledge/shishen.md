# 十神组合场景化

十神组合不能只说"杀印相生贵格"，须说明在人生中的具体应事方向：

| 组合 | 应事场景 | 不应事场景 |
|------|---------|-----------|
| 杀印相生 | 压力大的环境中反而能学习成长；被挑战时发挥最好 | 轻松安逸的环境中难以发力；不适合自由散漫的岗位 |
| 伤官配印 | 技术/学术领域有天赋；以专业技能立足 | 纯管理岗或纯执行岗（需要创造空间） |
| 财官相生 | 资源换地位；在商业环境中攀升 | 体制内完全按资历排队的岗位（需要市场验证） |
| 食伤生财 | 才华变现；创意/内容/咨询类工作 | 固定流水线或按部就班的工作体制 |
| 比劫争财 | 适合合伙但有分配风险 | 独享型商业模式（需要分钱机制） |
