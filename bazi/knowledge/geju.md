# 子平真诠 — 格局论命

## 核心原则

格局以月令为提纲，月令本气透干则定格，不透则以月令本气为虚格。格局以清纯为贵，杂浊为贱。

## 定格规则

月令支藏干透干定格局：

1. **透干优先**：月令本气→中气→余气，第一透干者定格
2. **都不透**：取月令本气为虚格，标注"格不显"
3. **多透**：以本气为主，中气余气为辅

## 用神与相神

| 概念 | 定义 |
|------|------|
| 用神 | 命局中最关键的十神，用于维护格局平衡 |
| 相神 | 辅助用神的十神，如正官格中的印星（官印相生） |
| 喜神 | 生扶用神的五行 |
| 忌神 | 克破用神的五行 |

## 判断要点

1. **清浊**：用神一位为清，多用神或用神受冲克为浊
2. **有力无力**：用神在地支有根、得月令生扶为有力
3. **位置**：用神与日主距离越近作用越大

## 十神组合场景化

十神组合不能只说"杀印相生贵格"，须说明在人生中的具体应事方向：

| 组合 | 应事场景 | 不应事场景 |
|------|---------|-----------|
| 杀印相生 | 压力大的环境中反而能学习成长；被挑战时发挥最好 | 轻松安逸的环境中难以发力；不适合自由散漫的岗位 |
| 伤官配印 | 技术/学术领域有天赋；以专业技能立足 | 纯管理岗或纯执行岗（需要创造空间） |
| 财官相生 | 资源换地位；在商业环境中攀升 | 体制内完全按资历排队的岗位（需要市场验证） |
| 食伤生财 | 才华变现；创意/内容/咨询类工作 | 固定流水线或按部就班的工作体制 |
| 比劫争财 | 适合合伙但有分配风险 | 独享型商业模式（需要分钱机制） |
