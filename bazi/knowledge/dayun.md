# 大运分析方法论

## 大运与命局的互动

每步大运须从三个维度分析：

**大运干支与命局的关系**
- 大运天干与四柱天干：生合冲克刑
- 大运地支与四柱地支：六合三合六冲六害
- 重点关注：大运冲合原局哪个柱，该柱对应的宫位和十神

**大运对用神的作用**
- 大运生扶用神 → 得运，该运有利
- 大运克制用神 → 失运，该运不利
- 大运助忌神 → 该运需谨慎

## 十年分治

每步大运不能笼统概括，须细分：

**前五年（天干主导）** — 大运天干的力量在五年中逐步增强，前五年受天干影响更大
**后五年（地支主导）** — 大运地支的力量在后五年显现，地支冲合的影响在后五年更明显
**换运年** — 前后两步大运交接之年（前后各一年），是人生节奏切换的关键节点，工作/感情/健康容易出现重大变化

## 大运与流年的配合

大运定十年基调，流年看当年应事：
- 流年生扶大运 → 该年趋势顺利
- 流年克制大运 → 该年有波折
- 流年与大运配合引动用神 → 该年为吉年
- 流年与大运配合引发忌神 → 该年为凶年
