# 大运分析方法论

## 大运与命局的互动

每步大运须从三个维度分析：

**大运干支与命局的关系**
- 大运天干与四柱天干：生合冲克刑
- 大运地支与四柱地支：六合三合六冲六害
- 重点关注：大运冲合原局哪个柱，该柱对应的宫位和十神

**大运对用神的作用**
- 大运生扶用神 → 得运，该运有利
- 大运克制用神 → 失运，该运不利
- 大运助忌神 → 该运需谨慎

## 十年分治

每步大运不能笼统概括，须细分：

**前五年（天干主导）** — 大运天干的力量在五年中逐步增强，前五年受天干影响更大
**后五年（地支主导）** — 大运地支的力量在后五年显现，地支冲合的影响在后五年更明显
**换运年** — 前后两步大运交接之年（前后各一年），是人生节奏切换的关键节点，工作/感情/健康容易出现重大变化

## 大运与流年的配合

大运定十年基调，流年看当年应事：
- 流年生扶大运 → 该年趋势顺利
- 流年克制大运 → 该年有波折
- 流年与大运配合引动用神 → 该年为吉年
- 流年与大运配合引发忌神 → 该年为凶年

## 应期推断

同一方法适用于六亲、财运、事业、健康、感情等所有领域。

### 第一步：确定目标

| 目标 | 对应 |
|------|------|
| 父亲 | 偏财星 + 年柱 |
| 母亲 | 正印星 + 年柱 |
| 配偶 | 男命正财/女命正官 + 日支 |
| 子女 | 食伤星 + 时柱 |
| 财运 | 正偏财星 + 财帛宫 |
| 事业 | 官杀星 + 官禄宫 |
| 健康 | 日主 + 忌神 + 疾厄宫 |

### 第二步：大运引动

大运给时间窗口。大运的如下行为打开应期窗口：

- **冲**：冲目标星或目标宫位 → 该运为该领域的窗口
- **合**：合住目标星 → 受牵制；合化 → 性质改变
- **克**：大运干支克制目标星 → 凶应窗口
- **生扶**：大运生扶目标星 → 吉应窗口
- **值**：大运干支与目标星/宫位完全相同 → 最强烈的引动

### 第三步：流年应事

流年定具体时间。如下情况为强应期：

- **双重引动**：大运+流年同时引动同一目标 → 强应年
- **三重引动**：大运+流年+命局自刑冲 → 极强年
- **换运叠加**：换运年前后一年+流年引动 → 特殊窗口
- **伏吟**：流年与目标宫位相同（年柱伏吟→父母事）

### 四类场景的信号强度

**六亲应期**：大运冲年柱 + 流年冲母星/父星 + 换运年叠加 → 强信号
**财运应期**：大运生扶财星 + 流年引动财帛宫 → 强信号
**事业应期**：大运引动官杀 + 流年合官杀 → 变动期
**健康应期**：大运+流年同时冲克日主或用神 → 需注意

### 应期精度

- 流年定年，流月定事
- 换运年（前一年交运+后一年）= 重点窗口
- 换运年叠加流年冲刑 = 重大变化概率高
