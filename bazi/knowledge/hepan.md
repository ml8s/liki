# 八字合盘方法论

## 核心理念

合盘不是两项逐项打分求和，而是先排除致命问题再看匹配度，最后评估可持续性。

## 分析流程

### 第一关：致命排除

**年柱冲** — 年柱代表根基，寅申冲、卯酉冲、巳亥冲等六冲为根基不稳。有则标记"年冲"，进入慎配/不利区间。
**日支冲** — 日支（夫妻宫）对冲，婚姻主矛盾多、磨合期长。子午冲、丑未冲等。
**配偶星空亡** — 男命正财落空亡、女命正官落空亡，对配偶的满意度低。
**日干纯克无通关** — 日主五行相克（如庚金克甲木），且八字中无通关之字（如乙木→火通关金木之战）。

以上四项任一成立，评级上限降为"慎配"，两项以上为"不利"。

### 第二关：匹配评估

**1. 日干关系**
- 日干相生（如甲木生丙火）：关系融洽，有付出意愿。男日主生女日主→男方主动付出型
- 日干相合（如甲己合）：情缘深，吸引力强
- 日干比和（同为金/木等）：平等型，但容易互不相让
- 日干纯克：有张力，需看克是否有制

**2. 夫妻宫互动**
- 六合（子丑合、寅亥合等）+ 三合（申子辰等）：良配，情缘深厚
- 六冲（子午冲、丑未冲等）：摩擦多，需看其他方面能否弥补
- 六害（寅巳害、卯辰害等）：暗中不和，小事积累
- 相刑（寅巳申互刑等）：口舌是非多

**3. 五行互补度**
- 双方八字五行放在一起，各五行都有分布，无某一行过旺造成偏枯
- 一方缺的五行另一方刚好有——加分
- 同一五行双方都过旺——减分（同五行过旺为"斗"）

**4. 十神传递**
- 看能量在两人之间的流动方向，是否形成良性循环
- 杀印相生传递：一方官杀旺（压力方）+ 另一方印旺（包容方）→ 压力被消化
- 财官相生传递：一方财旺（创造方）+ 另一方官旺（守成方）→ 互补
- 比劫相斗：双方比劫都旺 → 争夺型关系

### 第三关：可持续性

**5. 大运同步**
对比双方大运走势：
- 走势一致（同走吉运或同走凶运）：生活节奏同步，关系稳定
- 一方上运一方下运：容易不同频，需更多包容
- 起伏相关（上运时间错开）：互补型节奏，一方低谷时另一方支撑

**6. 紫微验证（可选）**
- 双方命宫星曜关系：相生为佳，相克为摩擦
- 夫妻宫星曜能否呼应
- 四化是否有交集

### 综合评级

| 评级 | 条件 |
|------|------|
| 良配 | 无致命问题，日干相生或相合，夫妻宫合多冲少，五行互补 |
| 可配 | 有一项匹配弱（如日干比和或夫妻宫冲），但其他方面弥补 |
| 慎配 | 有一项致命问题（年冲/日支冲/配偶星空亡），或两项以上匹配弱 |
| 不利 | 两项以上致命问题，或日干纯克无制+夫妻宫冲+大运不同步 |

评级不是"能不能成"，而是"需要付出多少额外努力"。结论须说明优势与风险点。
