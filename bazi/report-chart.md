# 八字排盘报告模板

基于 `bazi.chart` 返回数据生成八字报告。只引用实际存在的字段，不编造。

## 数据来源

- `bazi.chart` → 四柱（年/月/日/时）、日主、十神、纳音、神煞、藏干、大运起运时间
- `bazi.yongshen` → 五行计数、旺相休囚死、身强身弱、扶抑/调候/格局三派用神、喜神、忌神（LLM 综合判断最终用神）
- `bazi.hehui` → 天干五合、地支六合、三合三会、六冲、六害、六刑
- `bazi.chart_extra` → 胎元、命宫、身宫、拱夹、纳音生克、长生十二宫

## 报告结构

### 一、命盘总览

四柱排列表（含藏干），日主五行、身强身弱、格局。

### 二、五行与十神分析

五行分布（计数+强弱）、十神配置。日主与月令关系，生克制化格局。

### 三、用神喜忌

来自 `bazi.yongshen`。列出扶抑、调候、格局三派各自的用神/喜神/忌神，LLM 综合三派结果给出最终用神判断及理由。引用《滴天髓》特例时注明（如丙火见壬癸之别、庚金得火而锐等）。

### 四、大运走势

起运年龄、当前所处大运、每十年大运干支及简要提示。

### 五、当前流年

当年干支与命局关系，重点关注事项。
