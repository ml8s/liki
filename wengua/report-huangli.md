# 黄历择日报告模板

基于 `huangli.date`/`huangli.month` 或 `huangli.bond.date`/`huangli.bond.month` 返回数据生成择日报告。只引用实际存在的字段，不编造。

## 数据来源

- `huangli.date` / `huangli.month` → 宜忌、神煞、吉凶等级
- `huangli.bond.date` / `huangli.bond.month` → 八字合参后的宜忌、神煞、命主适合度

## 报告结构

### 一、结论（结论先行）

**明确推荐吉日**，如：
- "本月XX日最宜，六合吉日，不冲命主。"
- "本月无特别理想的吉日，XX日为次选，宜改至下月。"

### 二、推荐日期

- 列出 1-3 个推荐日
- 每个日期标注：公历/农历、星期、当日吉凶等级

### 三、宜忌概要

- 当日宜/忌与用户事项的匹配度
- 关键神煞（天德、月德、天赦等吉神；五虚、九空等凶煞）

### 四、八字合参说明

若为八字合参择日：
- 命主喜忌参考
- 推荐日是否与命主相合/相冲

### 五、排除说明

- 简单说明排除了哪些明显不好的日期（如月破、三煞日等）
