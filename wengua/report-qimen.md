# 奇门遁甲报告模板

基于对应 API 返回数据生成奇门报告。

## 数据来源

字段含义参考 `rpc.discover` 返回的 schema。只引用实际存在的字段，不编造。

## 报告结构

### 一、断语（结论先行）

**一句话明确判断**所问之事的方向/策略/局势判断。如：
- "奇门盘局显示当前时机有利，开门逢值使，宜主动出击。"
- "局中逢死门+白虎，事有阻碍，宜静待时机。"
- "天芮星临宫，此事需先解决内部问题再行动。"

### 二、值符值使

- 值符落宫、值使落宫
- 值符与问事事体的关系

### 三、三盘格局

- 天盘（九星）吉凶格局
- 人盘（八门）吉凶格局
- 神盘（八神）吉凶格局
- 重点看值符值使所在宫及问事宫位

### 四、方向与时机

- 有利方向（奇门方位）
- 时机判断：时家/日家/月家/年家格局
- 若问具体方向，指出吉方

### 五、策略建议

- 根据盘局给出可操作建议
- 注意避开凶门凶星所临方位
